## 📖 Project Overview

This project demonstrates an enterprise-grade Selenium automation framework
built using Java, Selenium WebDriver, TestNG, Maven, and the
Page Object Model (POM).

The framework automates functional and regression testing of the
ParaBank web application while following industry best practices
including reusable page objects, reporting, logging, and CI/CD
automation using GitHub Actions.

## ⭐ Features

- Selenium WebDriver Automation
- Java 21
- TestNG Framework
- Page Object Model (POM)
- Maven Build Management
- Extent HTML Reporting
- Screenshot Capture on Failure
- Logging Framework
- GitHub Actions CI/CD
- Automatic Build Status Badge
- Cross-platform Driver Management using WebDriverManager
